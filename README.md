# Datagaia

Marketing website for Datagaia — geomarketing, predictive analytics & programmatic advertising.

Single-page static site (`index.html`). No build step required.

## Deploy
Static site — deploys to Vercel with zero configuration (root `index.html`).

## Local preview
Open `index.html` in any browser, or run:

    python3 -m http.server 8000
